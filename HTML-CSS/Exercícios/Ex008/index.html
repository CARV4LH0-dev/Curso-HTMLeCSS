Principais formatações

Negrito/Destaque

Nessa frase, temos um termo em negrito usando a tag "b" (não semântica)

Nessa frase, temos um termo em destaque usando a tag "strong" (semântica)

Itálico/Ênfase

Nessa frase, temos um termo em itálico usando a tag "i" (não semântica)

Nessa frase, temos um termo em ênfase usando a tag "em" (semântica)

Texto marcado

Você pode criar um texto marcado usando a tag "mark" (não semântica)

Texto grande e pequeno

Você também pode criar um texto grande e um texto pequeno usando as tags "big" e "small", respectivamente ("big" = não semântica) ("small" = semântica)

Obs: a tag "big" é obsoleta no HTML5!

Texto deletado

Você também pode marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag "del" (não semântica)

Texto inserido

Você também pode marcar um texto como inserido para dar um sublinhado (e indicar que ele foi adicionado depois) usando a tag "ins" (semântica)

Você também pode marcar um texto com sublinhado usando a tag "u" (não semântica)

Texto sobrescrito

VocÊ também pode inserir coisas do tipo: x2+3 usando a tag "sup"

Texto subscrito

Você também pode inserir coisas do tipo: H2O usando a tag "sub"
